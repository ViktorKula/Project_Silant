### Backend starting (console 1):

```
cd backend
```
```
python -m venv venv
```
```
.\venv\Scripts\activate
```
```
pip install -r requirements.txt
```
```
python manage.py runserver
```

### Fronend starting (console 2):
```
cd frontend
```
```
npm install
```
```
npm start
```
